RH- Buffet Fenix

Olá! Seja bem vindo ao RH - Buffet Fenix - Projeto Acelera Pleno Atos, desenvolvido por Luiz Generoso, Danilo Azevedo, Leonardo Salvador e Damaris Santana.
Esse é um sistema para gerenciamento de pagamentos, tendo como príncipio básico administrar o RH de forma simples, fornecendo uma agilidade ao efetuar o pagamento aos colaboradores.
desenvolvido em Angular 13 e Spring Boot 2.
O sistema utiliza em seu Backend arquitetura de microserviços (Eureka Server, Discovery Server, Config Server, Oauth Server) implementado em sua segurança autenticação JWT Oauth 2.0
Em seu Frontend utilizamos biblioteca do framework PrimeNG.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais.
O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
